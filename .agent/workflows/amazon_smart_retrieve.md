# Amazon Image Retrieval & Smart Placement

This workflow explains how the new "Amazon Smart Retrieve" feature works and how "deformed hands" are prevented.

## 1. Amazon Smart Retrieve

When you input an Amazon product URL into Postgenius Pro:

1.  **Detection**: The system automatically detects that the input is an Amazon link.
2.  **Auto-Switch**: Even if your settings are set to "AI Generated Images" for products, the system will **automatically prioritize** fetching real product images from Amazon (using the Product Advertising API).
3.  **Retrieval**: It fetches the primary high-resolution image and any available variants.
4.  **Integration**: These real images are used instead of AI-generated ones for the product, ensuring 100% accuracy and realism.

## 2. Intelligent AI Placement

The AI writer has been upgraded to understand where to place these images:

1.  **Context Awareness**: As the AI writes the article (review, roundup, or guide), it identifies the best spots to show the product.
2.  **Placeholder Injection**: The AI inserts `[PRODUCT_IMAGE_URL_X]` placeholders naturally within the text (e.g., after a product description or verdict).
3.  **Rendering**: These placeholders are automatically replaced with the actual Amazon images (or AI-generated ones if Amazon fails) during the final HTML generation.

## 3. "No More Deformed Hands" (AI Image Fix)

For images that *must* be generated by AI (like hero images, step-by-step guides, or non-Amazon products):

1.  **Strict Prompt Engineering**: We have updated the image generation prompts in both Gemini and Free Tier services.
2.  **"No Hands" Policy**: The prompts now explicitly include strict negative constraints:
    *   `CRITICAL: NO hands, NO fingers, NO people, NO human body parts visible.`
    *   `The object must be on a surface, NOT held.`
3.  **Composition Control**: We force the AI to compose shots where objects are resting on tables, desks, or plates, eliminating the need for hands to appear (and thus be deformed).

## How to Verify

1.  **Amazon Link**: Paste an Amazon product URL into the input field.
2.  **Generate**: Watch as the system fetches real product images.
3.  **Review**: Check the generated article. You should see real Amazon images placed intelligently within the text.
4.  **AI Images**: Check the Hero image or Step images. They should be clean, professional, and free of distorted hands.
