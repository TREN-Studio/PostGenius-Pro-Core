## 🛠️ Community Edition Setup

This repository contains the **Community Edition** of the PostGenius Pro frontend and core logic. It has been stripped of proprietary TREN Studio API keys and production secrets, making it safe and easy for developers to learn from or run locally.

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/TREN-Studio/PostGenius-Pro-Core.git
   ```
2. Install dependencies:
   ```bash
   cd PostGenius-Pro-Core
   npm install
   ```
3. Set up your environment:
   Copy the example environment file and fill in your own credentials:
   ```bash
   cp .env.example .env
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
